# Chaotic Latent Space Sampler

Deterministic sampling of autoencoder latent spaces using
data-driven piecewise-linear chaotic maps.

This repository accompanies the paper:

**Deterministic Sampling of Autoencoder Latent Spaces via Data-Driven  
Piecewise-Linear Chaotic Maps**

## Overview

This code implements a fully deterministic generative mechanism operating
directly in learned latent spaces. The method constructs context-dependent,
piecewise-linear chaotic maps whose invariant probability densities locally
match empirical latent statistics.

Unlike diffusion or flow-based approaches, the sampler:
- does not require stochastic noise,
- does not learn vector fields,
- does not assume a Gaussian prior.

New latent samples are generated purely by deterministic iteration.

## Method summary

1. Each latent dimension is discretized into `B` coarse bins.
2. Conditional fine histograms are estimated from the remaining dimensions.
3. Piecewise-linear chaotic maps are synthesized via an inverse
   Frobenius–Perron construction.
4. Latent samples are generated by deterministic iteration.

The method can be used in:
- **single-seed mode** (trajectory of a single orbit),
- **multi-seed mode** (ensemble-based sampling for distribution matching).
